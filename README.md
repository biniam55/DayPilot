# DayPilot - AI-Powered Daily Planner

DayPilot is a modern, high-performance daily schedule assistant built to help users optimize their productivity using Generative AI.

## 🚀 Tech Stack

- **Next.js 15 (App Router)**: High-performance framework.
- **React 19**: Latest UI patterns.
- **Tailwind CSS & ShadCN UI**: Beautiful, responsive design.
- **Genkit 1.x**: AI integration with Gemini 2.5 Flash.

## 🛠 Local Setup (Cursor)

1.  **Clone/Download**: Extract the project files to a local folder.
2.  **Install**: `npm install`
3.  **Environment**: Create a `.env` file with your keys (Firebase/Genkit).
4.  **Run**: `npm run dev`

## 📦 Pushing to GitHub

If you want to save your progress or deploy, follow these steps:

1.  **Initialize Git**:
    ```bash
    git init
    git add .
    git commit -m "Initialize DayPilot project"
    ```
2.  **Link to GitHub**:
    - Create a new repository on [GitHub](https://github.com/new).
    - Copy the repository URL.
    - Run: `git remote add origin YOUR_REPOSITORY_URL`
3.  **Push**:
    ```bash
    git branch -M main
    git push -u origin main
    ```
