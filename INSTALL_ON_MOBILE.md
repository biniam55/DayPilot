# 📱 Install DayPilot on Your Phone

## Quick Installation Guide

### iPhone/iPad (iOS)

1. **Open Safari** (must use Safari, not Chrome)
2. **Visit** your DayPilot URL
3. **Tap** the Share button (⬆️) at the bottom
4. **Scroll down** and tap "Add to Home Screen"
5. **Tap "Add"**
6. **Done!** Open from your home screen

### Android

1. **Open Chrome** (or any browser)
2. **Visit** your DayPilot URL
3. **Tap** the menu (⋮) in the top right
4. **Tap** "Add to Home screen" or "Install app"
5. **Tap "Add"** or "Install"
6. **Done!** Open from your home screen

## Troubleshooting

### iOS Issues:

**"Add to Home Screen" not showing?**
- Use Safari (not Chrome)
- Make sure you're on the website
- Try refreshing the page

**App opens in Safari?**
- Delete from home screen
- Clear Safari cache
- Add again

### Android Issues:

**No install prompt?**
- Look for "Add to Home screen" in menu
- Try Chrome browser
- Check if already installed

**App not working offline?**
- Open app while online first
- Let it fully load
- Then try offline
